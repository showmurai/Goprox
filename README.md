Goprox
======

Goprox is a UDP proxy server by Go language.

Usage
-----

Please change receivable and sendable IP-address and port.

    AddrFromClient = "RECEIVABLE"
    TARGSRV        = "SENDABLE"

After type go-run command.

    $ Go run GoproxUDP.go

or build executable file.

    $ Go build GoproxUDP.go

    $ ./GoproxUDP
